Python 2.7 must be installed for the spy to work.

To compile, simply use call "make".

To run the spy, run the runSpy.sh file: ./runSpy.sh

Some fiddling with constants may be required to tune readings on different CPUs.

Three files will be created each time the filter runs:

'output.csv', 'filtereddata.csv', and 'results.txt'

The csv files are for the use of the filter.

'results.txt' stores the strings read from the channel
